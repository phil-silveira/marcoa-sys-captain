WSTK - Thermal Control System
Sistema de Monitoramento e Controle para piso aquecido

O WSTK - Thermal Control System é um sistema que foi projetado e desenvolvido pelas empresas MarcoA e GKW Agro Tech. A MarcoA Sistemas de Aquecimento é uma empresa especializada em projeto, fabricação e instalação de cabos de aquecimento para pisos. A GKW Agro Tech é uma empresa especializada no desenvolvimento de soluções em tecnologia da informação para o Agronegócio.

O WSTK surgiu a partir da necessidade do mercado de um sistema inovador para o monitoramento e controle de ambientes que possuem piso aquecido. Este sistema é moderno, especializado e inteligente que suporta o gerenciamento de temperatura individual para cada ambiente a partir de smartphones, tablets ou computadores propiciando assim total conforto e economia real de energia elétrica para seu usuário.
